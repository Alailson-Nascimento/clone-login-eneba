# 🎮 Clone da Página de Login - Eneba

> ⚠️ **AVISO IMPORTANTE:** Este projeto é uma **cópia para fins de estudo e portfólio**.  
> Não possui fins comerciais. Todos os direitos de imagem, marca e design pertencem à [Eneba](https://www.eneba.com/).

---

## 🎯 Objetivo

Este projeto foi desenvolvido como **Avaliação Parcial 1** de um curso de desenvolvimento web.  
O objetivo é demonstrar a capacidade de **reproduzir uma interface profissional** passada por um Web Designer/UI-UX, servindo como **portfólio** para apresentação das minhas habilidades como desenvolvedor front-end.

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estruturação semântica da página |
| CSS3 | Estilização, Flexbox e Media Queries |
| JavaScript | Interatividade (alternância de estados de login) |
| Font Awesome | Ícones sociais |
| Git/GitHub | Versionamento e hospedagem do código |

## 🚀 Como executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/Alailson-Nascimento/clone-login-eneba.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd clone-login-eneba
   ```
3. Abra o arquivo `index.html` no navegador.

> 💡 **Dica:** Você também pode usar a extensão **Live Server** do VS Code para visualizar em tempo real.

---

## 📚 O que aprendi

- ✅ Estruturação semântica com tags HTML5 (`header`, `main`, `section`, `footer`)
- ✅ Layouts responsivos com **Flexbox**
- ✅ **Media Queries** para adaptação mobile/desktop
- ✅ Manipulação do DOM com JavaScript puro
- ✅ Alternância de estados na interface (login com link mágico ↔ login com senha)
- ✅ Versionamento de código com Git e publicação no GitHub

---

## 📝 Observações

Este é um projeto de **estudo e aprendizagem**. Algumas diferenças em relação ao site original são intencionais ou resultado do processo de desenvolvimento:
- Ícones sociais utilizam Font Awesome em vez de imagens originais
- Layout simplificado para foco nos conceitos fundamentais
- Funcionalidades de backend (autenticação real) não implementadas
